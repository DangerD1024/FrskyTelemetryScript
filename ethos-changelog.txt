- added support for Ethos on X10/X12
- full screen layout for X10,X12,X18 and X20
- external serial port support on x18 and x20 (MavToPT)
- maps support

Ethos version 1.0.0c dev, all radios

- fixed timer
- fixed widget reset
- fixed x10 messages screen
- fixed alt display when terrain enabled
- added 16s (cells) support

Ethos version 1.0.0c-dev, all radios

- fixed incompatibility with external protocol lua scripts (Ghost,MultiModule)

Ethos version 1.0.0c dev, all radios

- hide arming status on messages and map
